# Travel

开发工程师旅游指南，更新中(Programmer's Travel Guide, being updated)

本项目致力打造一个属于开发工程师的旅行✈️指南。

🌟项目从中国34个一级行政区(包括23个省[含台湾省]、5个自治区、4个直辖市以及2个特别行政区)开始逐级按照省、市、区、景点以及推荐路线进行介绍。

🌟内容涵盖目的地的人文地理区域概况介绍和景点介绍。

🌟程序员脱单蜜月神器！！
